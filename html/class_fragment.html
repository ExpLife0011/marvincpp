Detailed Description

A Fragment is conceptually a sub buffer of an MBuffer, it consists of a pointer that lies within the address range of the memory managed by an MBuffer and a size which keeps within the MBuffers memory slab.

But the Fragment class itself does not know about the underlying MBuffer. That is reserved for the FBuffer class

std::size_t  len 
)

addToEnd - adds the block of memory p to p + len - 1 to the end of the current fragment.

Throws error if endPointer() + 1 != p : that is if the new memory is not contiguous with the existing memory.

The caller must ensure that the full address range of the new piece is within the address range of the underlying MBuffer.
